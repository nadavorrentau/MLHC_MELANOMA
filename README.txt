Preprocessing, Classification Model and analysis between classification correctness and segmentation results are in the ipynb notebook. 
Output is in the notebook, except for Grad-CAM images as they were to heavy to upload to git. See some examples in the appendix of the report.


preprocessing.py - preprocessing for the segmentation model
segmentation.py - segmentation model training and evaluation
metrics.py - metrics and loss functions for segmentation
exceptions.py - custom exceptions
tuning - tuning model hyperparameters
tta - test time augmentation (detailed in our paper)
